# 05-UI-DESIGN-BRIEF.md
## Design Goal
Cool, minimal, modern Notion-like UI but more “personal productivity vibe”.

## Visual Style
- Theme: soft dark + light toggle
- Neutral base + 1 accent color (user change later)
- Large whitespace, rounded cards, subtle shadows
- Typography:
  - Title: bold, 20–24px
  - Body: 14–16px clean sans
- Icon system: lucide / feather style

## Layout
### Sidebar
- Left fixed sidebar
- Page tree expandable
- Quick action buttons:
  - + Page
  - + Learning
  - + Reminder

### Dashboard
- 3 main cards stacked or grid:
  1. Today Reminders
  2. Learning Priority Queue
  3. Recent Pages

### Pages Editor
- Top bar: title + icon + breadcrumb
- Blocks in clean column, wide reading area
- “/” command menu (optional MVP)
- Floating “+ block” button

### Learning Planner
- List view with filters on top:
  - Priority
  - Status
  - Date range
- Each row card:
  - Title
  - Priority pill
  - Status
  - Date + duration
- Detail drawer right side:
  - title edit
  - meta fields
  - subpoints list

### Software List
- Category tabs left/top
- Search bar
- Cards showing:
  - name
  - tags
  - link
  - short note

## Figma Deliverables
- Design system (colors, type, spacing, buttons)
- Components library (sidebar, cards, pills, inputs)
- Screens (Desktop + Mobile):
  - Dashboard
  - Pages
  - Learning list + detail
  - Software list
  - Reminders panel
